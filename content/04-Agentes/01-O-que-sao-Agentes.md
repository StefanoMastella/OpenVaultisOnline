# O que são Agentes?

## Conceito Simples

**Agentes** são inteligências artificiais especializadas que trabalham com você no seu Vault.

Pense neles como:
- **Assistentes especializados** que entendem seu contexto
- **Colegas de trabalho** que nunca esquecem nada
- **Amplificadores** da sua capacidade de processar informação

## Como Funcionam

Os Agentes:
1. **Lêem** seus Canvas e notas
2. **Entendem** o contexto e conexões
3. **Sugerem** expansões e melhorias
4. **Criam** conteúdo novo baseado no que você já tem

## Auto-Routing (Roteamento Automático)

O sistema tem **11 Agentes Especializados** que são ativados automaticamente baseado no contexto:

- **Currency God** - Para questões financeiras
- **Ogma Sonosfera** - Para projetos acústicos/engenharia
- **Stéfano Don Juan** - Para relacionamentos
- **Dracomante** - Para tarot/lenormand
- **SOLO Codador** - Para questões técnicas
- **Girotto** - Agente principal (duplo digital)
- **GitHub Triage** - Para bugs e soluções
- **Sinapse Architect** - Para organização de vault
- **Mercador Malkhut** - Para negócios
- **Sherlok** - Para investigação/análise
- **Ipê News Journalist** - Para curadoria de notícias

## Fuzzy Matching

O sistema é inteligente o suficiente para detectar o Agente certo mesmo se você digitar o nome errado (ex: "Agma" → Ogma).

## Como Usar

### Método 1: Automático

Simplesmente faça uma pergunta ou peça ajuda. O sistema detecta o contexto e roteia para o Agente certo.

### Método 2: Explícito

Mencione o nome do Agente ou use `@nome-do-agente` se disponível.

### Método 3: Mostrar Canvas

Mostre um Canvas ao Agente e peça para expandir, analisar ou sugerir melhorias.

## Exemplos Práticos

**Exemplo 1 - Financeiro**:
```
"Posso comprar este item com meu orçamento atual?"
→ Sistema roteia para Currency God automaticamente
```

**Exemplo 2 - Projeto Técnico**:
```
"Analise este Canvas de arquitetura de software"
→ Sistema roteia para SOLO Codador
```

**Exemplo 3 - Relacionamento**:
```
"Como devo responder esta mensagem?"
→ Sistema roteia para Stéfano Don Juan
```
