# Como Trabalhar com Agentes

## Fluxo Básico

1. **Crie** seu Canvas ou notas
2. **Mostre** para o Agente
3. **Peça** o que você precisa
4. **Revise** e integre as sugestões

## Dicas para Melhores Resultados

### 1. Seja Específico

❌ "Ajude-me com este projeto"
✅ "Analise este Canvas de viagem e crie um roteiro detalhado com restaurantes e atrações"

### 2. Forneça Contexto

- Mostre o Canvas completo
- Mencione objetivos ou restrições
- Compartilhe informações relevantes

### 3. Itere

- Revise o que o Agente criou
- Peça refinamentos
- Ajuste conforme necessário

### 4. Use Canvas para Contexto

Canvas são especialmente poderosos porque:
- Mostram estrutura visual
- Revelam conexões entre ideias
- Fornecem contexto rico para o Agente

## Exemplos de Comandos

### Para Expandir Ideias
```
"Olhe este Canvas sobre [tópico]. 
Expanda cada ideia com detalhes práticos e próximos passos."
```

### Para Analisar
```
"Analise este Canvas e identifique:
- O que está faltando
- O que pode ser melhorado
- Sugestões de organização"
```

### Para Criar Conteúdo
```
"Baseado neste Canvas, crie:
- Um plano de ação detalhado
- Uma lista de recursos necessários
- Um timeline com marcos"
```

### Para Refinar
```
"Refine este Canvas aninhado com:
- Informações mais específicas
- Elementos acionáveis
- Links para recursos externos"
```

## Trabalhando com Canvas Aninhados

Quando você tem Canvas aninhados:
1. Mostre o Canvas principal primeiro
2. Depois mostre o Canvas aninhado específico
3. O Agente entende a hierarquia e contexto
