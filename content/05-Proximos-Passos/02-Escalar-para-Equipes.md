# Escalar para Equipes

## Segundo Cérebro Coletivo

Obsidian Synapses não é apenas individual - pode ser usado por **equipes e comunidades**.

## Como Funciona

### Vault Compartilhado

- Equipe usa o mesmo Vault (ou Vaults sincronizados)
- Cada membro tem acesso aos Canvas
- Agentes trabalham com contexto coletivo

### Canvas Colaborativos

- Múltiplas pessoas podem editar
- Cada um adiciona sua perspectiva
- Agentes sintetizam visões diferentes

### Agentes Especializados por Função

- Cada área da equipe pode ter Agentes especializados
- Exemplo: Time de vendas → Agente Mercador
- Exemplo: Time técnico → Agente SOLO Codador

## Casos de Uso em Equipe

### Planejamento de Projeto
- Canvas principal com visão geral
- Canvas aninhados por time/função
- Agentes para análise de dependências

### Brainstorming Coletivo
- Canvas aberto para ideias
- Todos adicionam contribuições
- Agentes sintetizam e organizam

### Documentação Viva
- Canvas que evolui com o projeto
- Agentes mantêm atualizado
- Histórico de mudanças visível

## Boas Práticas

1. **Estabeleça um Hub Principal**
   - Canvas "Comece Aqui" para a equipe
   - Links claros para diferentes áreas

2. **Use Convenções de Nomenclatura**
   - Padrão claro para nomes de Canvas
   - Facilita navegação e busca

3. **Documente Processos**
   - Canvas explicando como a equipe usa o sistema
   - Onboarding para novos membros

4. **Revise Regularmente**
   - Agentes podem sugerir melhorias
   - Equipe pode iterar sobre estrutura
