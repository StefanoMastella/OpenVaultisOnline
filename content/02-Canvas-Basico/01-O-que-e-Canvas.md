# O que é Canvas?

## Conceito Visual

**Canvas** = **Tela** (como a de pinturas)

No Obsidian, Canvas é uma **ferramenta de visualização** que permite organizar suas notas visualmente em um quadro infinito.

## Por que Canvas é Poderoso?

Muitos de nós somos **aprendizes visuais**. Poder manipular suas anotações de maneira visual ajuda a:
- Lembrar melhor das informações
- Ver conexões entre ideias
- Navegar, editar e visualizar tudo em um só lugar
- Fazer brainstorming visual

## O Diferencial do Canvas

O Canvas permite **agregar atomicidade com incrementalidade** - você pode:
- Colocar notas como se fossem post-its
- Mover e reorganizar livremente
- Ajustar aparência
- **E continuar usando todos os superpoderes do Obsidian**: links, tags, plugins

## Canvas como Injeção de Contexto

**Insight Chave**: Um Canvas é tão simples quanto um arquivo .md, mas funciona como **injeção de contexto** para modelos de IA (LLMs).

Quando você mostra um Canvas para um Agente, ele vê:
- Todas as notas conectadas
- A estrutura visual das ideias
- Os relacionamentos entre conceitos

Isso é o que chamamos de **Prompt 2D** - comunicação visual com inteligência artificial.
