# Links e Conexões no Canvas

## Criando Links

Os itens que você adiciona a um Canvas podem ser **conectados com links**:

1. Passe o mouse sobre qualquer item no canvas
2. Você verá um **círculo preenchido** em cada lado do item
3. **Clique e segure** no círculo
4. **Arraste** para:
   - Um item existente (cria link entre eles)
   - Uma área vazia (abre menu para criar novo item)

## Criando Novo Item a Partir de Link

Quando você arrasta um link para uma área vazia:
- Um menu será aberto perguntando o que criar
- Você pode escolher:
  - **Nova nota** (cria arquivo .md)
  - **Novo cartão** (cria card no canvas)
- Atualmente não há opção para adicionar mídia diretamente assim

## Rotulando Links

Você pode **rotular os links** entre itens:
- Clique no link (a linha entre os itens)
- Digite o rótulo desejado
- Isso ajuda a explicar o tipo de relação

**Exemplos de rótulos**:
- "leva a"
- "depende de"
- "exemplo de"
- "contradiz"

## Por que Links São Importantes?

No Obsidian Synapses, os links criam:
- **Conexões visuais** entre ideias
- **Contexto** para os Agentes entenderem relacionamentos
- **Navegação** fácil entre conceitos relacionados
- **Mapas mentais** que mostram como você pensa
