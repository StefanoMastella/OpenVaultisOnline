# Como Criar um Canvas

## Ativando o Canvas

O Canvas é um **plugin base** do Obsidian, então já vem instalado. Você só precisa ativá-lo:

1. Clique no ícone de **configurações** (⚙️) no canto inferior esquerdo
2. Selecione **Plugins Base** no menu à esquerda
3. No lado direito, ao lado de "Canvas", há um interruptor
4. Clique para **ativar** (se ainda não estiver ativo)

## Criando um Novo Canvas

### Método 1: Pelo Menu Lateral

1. No menu de ícones à esquerda, procure o ícone que parece uma **pavimentação** (ou grade)
2. Passe o mouse sobre ele - aparecerá "Create new canvas"
3. Clique no ícone
4. Um novo canvas será criado na aba ativa

### Método 2: Pelo Menu de Arquivo

1. Clique com botão direito em uma pasta
2. Selecione "New Canvas"
3. Dê um nome ao arquivo

## Nomeando seu Canvas

Quando criar um novo canvas:
- Um arquivo sem título será criado
- **Dê um nome descritivo** ao seu canvas
- O nome aparecerá no menu esquerdo com a palavra **"CANVAS"** ao lado
- Isso facilita identificar entre notas normais (.md) e canvas

## Abrindo um Canvas Existente

Canvas existentes são facilmente identificáveis:
- Têm o marcador **"CANVAS"** ao lado do nome do arquivo
- Você abre clicando no nome, como qualquer outra nota
