# Elementos do Canvas

Quando você cria um novo Canvas, aparecem instruções básicas na tela. Cada item pode ser um dos **três tipos de elementos**:

## 1. Cartões Livres (Cards)

**O que são**: Cartões em branco criados especificamente para aquele canvas.

**O que você pode colocar**:
- Texto
- Tarefas (checkboxes)
- Links

**Como criar**:
- Use o **primeiro botão** do menu inferior
- Ou clique duas vezes em uma área vazia do canvas
- Ou arraste o ícone do menu para o canvas

**Uso**: Ideal para anotações rápidas, ideias soltas, ou informações que não precisam ser arquivos separados.

## 2. Elemento de Notas

**O que são**: Notas existentes do seu vault adicionadas ao canvas.

**Característica importante**: 
- Ao editar uma nota dentro de um canvas, o arquivo original também é atualizado
- É a mesma nota, apenas visualizada no canvas

**Como criar**:
- Use o **segundo botão** do menu inferior
- Ou arraste uma nota do menu lateral para o canvas
- Ou arraste o ícone do menu para o canvas

**Uso**: Conectar notas relacionadas visualmente, criar mapas de conhecimento.

## 3. Elemento de Mídia

**O que são**: Arquivos já existentes no seu vault (imagens, PDFs, vídeos).

**Tipos suportados**:
- PDF
- MP4 (vídeo)
- JPEG, PNG (imagens)
- E outros formatos suportados pelo Obsidian

**Como criar**:
- Use o **terceiro botão** do menu inferior
- Ou arraste arquivos diretamente para o canvas
- Ou arraste o ícone do menu para o canvas

**Uso**: Ilustrar ideias, adicionar contexto visual, criar apresentações.

## Dica: Arrastar e Soltar

Você pode arrastar:
- **Notas** do menu lateral diretamente para o canvas
- **Imagens** de pastas diretamente para o canvas
- **Ícones do menu** para criar elementos específicos
