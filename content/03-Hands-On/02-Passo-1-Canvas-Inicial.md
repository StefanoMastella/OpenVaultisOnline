# Passo 1: Criar Canvas Inicial

## Objetivo

Criar um Canvas simples com as ideias básicas da sua viagem (ou projeto).

## Instruções

### 1. Criar o Canvas

1. Crie um novo Canvas chamado `Minha Viagem` (ou o nome do seu projeto)
2. Use o método que você aprendeu anteriormente

### 2. Adicionar Ideias Básicas

Crie **cartões livres** com informações iniciais:

**Para uma viagem, adicione**:
- Destino(s) que você quer visitar
- Datas aproximadas
- Orçamento estimado
- Pessoas que vão junto (se aplicável)

**Para um projeto pessoal, adicione**:
- Ideia principal
- Objetivos
- Recursos disponíveis
- Prazo (se houver)

### 3. Organizar Visualmente

- Mova os cartões para posições que façam sentido
- Agrupe ideias relacionadas próximas
- Não se preocupe em ser perfeito - isso é só o início!

### 4. Adicionar Algumas Conexões

- Conecte ideias relacionadas com links
- Rotule os links se quiser (ex: "requer", "influencia")

## Exemplo Visual

```
[Destino: Paris] ──requer──> [Orçamento: R$ 5.000]
         │
         └──requer──> [Datas: Julho 2025]
```

## Resultado Esperado

Você deve ter um Canvas com:
- 3-5 cartões com informações básicas
- Alguns links conectando ideias relacionadas
- Uma visão geral do que você quer fazer
