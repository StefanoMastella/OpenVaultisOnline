# Passo 2: Expandir com Agente

## O que é um Agente?

Um **Agente** é uma inteligência artificial que:
- Entende o contexto do seu Vault
- Pode ler seus Canvas
- Pode expandir suas ideias
- Trabalha como um assistente especializado

## Como Funciona a Injeção de Contexto

Quando você mostra um Canvas para um Agente:
1. O Agente **lê toda a estrutura visual**
2. **Entende as conexões** entre ideias
3. **Expande** com informações relevantes
4. **Sugere** próximos passos

Isso é o que chamamos de **Prompt 2D** - você comunica visualmente, não apenas com texto.

## Instruções

### 1. Abrir o Canvas no Agente

Se você está usando **Cursor** ou **Trae**:
1. Abra o Canvas que você criou
2. No chat do Agente, digite algo como:
   ```
   Olhe este Canvas sobre minha viagem. 
   Expanda as ideias e me dê sugestões de:
   - Roteiro detalhado
   - Dicas práticas
   - Checklist de preparação
   ```

### 2. Ver a Expansão

O Agente vai:
- Analisar seu Canvas
- Criar novos cartões com informações expandidas
- Sugerir conexões adicionais
- Criar um roteiro ou plano de ação

### 3. Integrar as Sugestões

- Revise o que o Agente criou
- Mova e organize os novos elementos
- Conecte com links se necessário
- Ajuste o que não fizer sentido

## O que Você Deve Ver

Seu Canvas agora deve ter:
- **Mais elementos** (cartões, notas)
- **Informações expandidas** (roteiro, dicas, checklist)
- **Conexões mais ricas** entre ideias
- **Próximos passos** sugeridos

## Insight Importante

Este é o **Aha Moment**:
- Você começou com ideias soltas
- O Agente transformou em um plano estruturado
- Tudo ficou registrado e reutilizável
- Você pode continuar expandindo infinitamente
