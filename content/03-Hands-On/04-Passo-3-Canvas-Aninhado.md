# Passo 3: Criar Canvas Aninhado

## O que é um Canvas Aninhado?

Um **Canvas Aninhado** (nested canvas) é um Canvas dentro de outro Canvas. É como criar uma "página de detalhes" de um conceito maior.

## Por que Usar Canvas Aninhados?

- **Organização**: Mantém o Canvas principal limpo
- **Detalhamento**: Permite aprofundar em tópicos específicos
- **Navegação**: Cria uma hierarquia visual de informações
- **Contexto**: Cada nível tem seu próprio contexto para Agentes

## Instruções

### 1. Identificar o que Detalhar

No seu Canvas expandido, escolha um tópico que precisa de mais detalhes:

**Para viagem**:
- Detalhes de cada destino
- Restaurantes e atrações
- Roteiro dia a dia

**Para projeto**:
- Tarefas específicas
- Timeline detalhada
- Recursos necessários

### 2. Criar o Canvas Aninhado

1. Crie um novo Canvas (ex: `Detalhes - Paris`)
2. Adicione este Canvas como **elemento de nota** no Canvas principal
3. Ou crie um link do tópico principal para o novo Canvas

### 3. Estruturar os Detalhes

No Canvas aninhado, adicione:
- Informações específicas do tópico
- Sub-tarefas ou sub-itens
- Links para recursos externos
- Imagens ou referências

### 4. Conectar com o Canvas Principal

- Certifique-se de que há um link claro entre o Canvas principal e o aninhado
- Isso permite navegação fácil
- E ajuda Agentes a entender a hierarquia

## Exemplo Visual

**Canvas Principal**:
```
[Viagem para Paris] ──detalhes──> [Canvas: Detalhes Paris]
```

**Canvas Aninhado (Detalhes Paris)**:
```
[Restaurantes] ──> [Atrações] ──> [Roteiro Dia 1]
```

## Resultado Esperado

Você deve ter:
- Canvas principal com visão geral
- Canvas aninhado com detalhes específicos
- Links claros entre eles
- Estrutura hierárquica visual
