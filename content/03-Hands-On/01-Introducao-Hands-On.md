# Hands-On: Meu Primeiro Segundo Cérebro

## O que vamos fazer?

Vamos criar um projeto prático do zero, passo a passo. Você vai:
1. Criar um Canvas simples com suas ideias
2. Ver como um Agente expande essas ideias
3. Criar um Canvas aninhado (nested) com detalhes
4. Ver o Agente refinar tudo

## Exemplo: Planejar Minha Viagem

Vamos usar o exemplo de **planejar uma viagem** porque é:
- Tangível e empolgante
- Fácil de entender
- Mostra o poder do sistema na prática

Mas você pode adaptar para:
- Um projeto pessoal
- Organizar um hobby
- Qualquer coisa que você queira planejar

## Estrutura do Hands-On

1. **Passo 1**: Criar Canvas inicial com ideias básicas
2. **Passo 2**: Injetar no Agente e ver a expansão
3. **Passo 3**: Criar Canvas aninhado com detalhes
4. **Passo 4**: Injetar nested e ver refinamento

## O que você vai aprender

- Como estruturar ideias visualmente
- Como trabalhar com Agentes
- Como criar Canvas aninhados
- Como o sistema transforma ideias soltas em planos concretos
